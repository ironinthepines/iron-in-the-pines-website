Method

Every class is built with intention.

01

Prepare

Start with a complete warm up that gets your joints, core, and nervous system ready to lift well.

02

Lift

Programming focuses on four primary compound lifts, which efficiently maximize strength, muscle growth, and functional fitness by engaging multiple joints and large muscle groups simultaneously.

03

Carry

Every class includes a loaded carry, which has full-body benefits and is a key movement contributing to longevity. Classes also fold in balance, jumping, dynamic work, and cardio bursts for practical, athletic strength.

04

Core

Inner core strength creates a strong, stable center for life's twists and turns, safe lifting, and a cool down to follow.
